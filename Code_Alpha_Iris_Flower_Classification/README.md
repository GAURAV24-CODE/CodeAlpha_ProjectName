# 🌸 Iris Flower Classification | CodeAlpha Data Science Internship

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Data Science Internship**. The objective is to build a Machine Learning model that accurately classifies Iris flowers into one of three species based on their physical measurements.

The Iris dataset is one of the most popular datasets for learning and demonstrating classification algorithms in Machine Learning.

---

## 🎯 Objective

Develop a classification model that predicts the species of an Iris flower using the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The target classes are:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## 📊 Dataset Information

The dataset contains **150 observations** with **4 numerical features** and **3 target classes**.

| Feature      | Description          |
| ------------ | -------------------- |
| Sepal Length | Length of sepal (cm) |
| Sepal Width  | Width of sepal (cm)  |
| Petal Length | Length of petal (cm) |
| Petal Width  | Width of petal (cm)  |

Target Variable:

* Setosa
* Versicolor
* Virginica
